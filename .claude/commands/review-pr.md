PR の差分をレビューします。

以下の手順で進めてください:

1. **差分取得**: レビュー対象のブランチ or PR 番号を確認する
   ```
   git diff main...HEAD
   # または
   gh pr diff {PR番号}
   ```

2. **チェックリスト確認**: `agent-skills/common/review-checklist.md` の全項目を確認する:
   - 機能・ロジック
   - コード品質
   - セキュリティ
   - テスト
   - ドキュメント
   - 運用

3. **レビュー結果レポート**: 以下の形式で報告する:
   ```
   ## レビュー結果

   ### ✅ 問題なし
   - （項目）

   ### ⚠️ 要確認
   - （項目）: （理由・提案）

   ### ❌ 要修正
   - （項目）: （問題・修正方針）

   ### 総合判定
   Approve / Request Changes / Comment
   ```

4. 修正が必要な場合は対応を実施する
